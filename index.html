Recently Added

New Fingerprint.js - Website

Fast browser fingerprint library. Written in pure JavaScript, no dependencies. By default uses Murmur hashing and returns a 32bit integer number. Hashing function can be easily replaced. Feather weight: only 843 bytes when gzipped.

New ES5-Shim - Website

ECMAScript 5 compatibility shims for legacy JavaScript engines.

New Converse.js - Website

A free and open-source XMPP chat client for your website.

New C3.js - Website

D3-based reusable chart library.
